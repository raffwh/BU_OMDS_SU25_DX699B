answer

Between boxplot and histogram, boxplot gives more statistical information such as the median line, the quartiles, and the extreme values. Histogram mostly gives the spread and the count of each event. However, boxplot may be preferred on academic research or people involved in statistics. Thus, histogram is usually used in most cases as more readers (or lay persons) can easily digest or understand the graph. To make it more interesting, we can always add median line, or color the extreme values in the histogram to give similar information as a boxplot.

def step2_numerics(df, columns, bins=30, figsize=(4,2)):
    """
    Generate several histograms for selected cols.
    
    Parameters:
    df (pd.DataFrame): The dataset.
    columns (list): List of column names to plot histograms for.
    bins (int): Number of bins for histograms (default=30).
    figsize (tuple): Figure size (default=(8, 4)).
    
    """
    for col in columns:
        plt.figure(figsize=figsize)
        sns.histplot(df[col], bins=bins, kde=True, edgecolor="black",  shrink=0.5)
        
        # Formatting
        plt.title(f"Histogram of {col}", fontsize=11, fontweight="bold")
        plt.xlabel(col)
        plt.ylabel("Frequency")
        plt.grid(axis='y', linestyle='--', alpha=0.5)
        
        plt.show()


def step2_discrete(df, columns, figsize=(4,2)):
    """
    Generate several histograms for selected cols.
    
    Parameters:
    df (pd.DataFrame): The dataset.
    columns (list): List of column names to plot histograms for.
    bins (int): Number of bins for histograms (default=30).
    figsize (tuple): Figure size (default=(8, 4)).
    
    """
    for col in columns:
        plt.figure(figsize=figsize)
        sns.histplot(df, x =col, discrete=True, edgecolor="black",  shrink=0.5)
        
        # Formatting
        plt.title(f"Histogram of {col}", fontsize=11, fontweight="bold")
        plt.xlabel(col)
        plt.xticks(sorted(df[col].unique())) 
        
        plt.ylabel("Frequency")
        plt.grid(axis='y', linestyle='--', alpha=0.5)
        
        plt.show()
